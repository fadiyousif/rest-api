# REST API

Built with Node & Express. Tested with Chai & Mocha.

## Routes

```
GET      /api/employees
POST     /api/employees
DELETE   /api/employees/:id

```

## Usage

```
# Install dependencies
npm install
yarn install

# Run in development
npm run dev
yarn run dev

# Run in production
npm start
yarn start

# Run tests
npm test
yarn test
```
